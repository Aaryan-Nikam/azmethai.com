# `azmeth tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
azmeth tui
azmeth tui --url ws://127.0.0.1:18789 --token <token>
azmeth tui --session main --deliver
```
