# `azmeth setup`

Initialize `~/.azmeth/azmeth.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
azmeth setup
azmeth setup --workspace ~/.azmeth/workspace
```

To run the wizard via setup:

```bash
azmeth setup --wizard
```
